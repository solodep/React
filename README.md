##  Описание
Разработан прототип SPA-приложения (React) для продажи молочной продукции.
Приложение включает:
- Авторизацию пользователей.
- Каталог товаров с пагинацией.
- Детальную страницу товара.
- Защиту маршрутов (доступ к товарам только после входа).

---

##  Как запустить проект

Для работы нужно запустить два терминала: для сервера и для клиента.

### 1. Серверная часть (Backend)
cd server
npm install
npm start
(Сервер запустится на http://localhost:3001)

### 2. Клиентская часть (Frontend)
cd client
npm install
npm run dev
(Сайт откроется на http://localhost:5173)

---

##  Функционал
1. **Кастомный хук useServerGoods:** Обрабатывает загрузку, ошибки и пагинацию.
2. **Маршрутизация:**
   - / (Главная)
   - /login (Вход)
   - /goods (Каталог - защищено)
   - /goods/:id (Товар - защищено)
3. **Build:** Проект собирается командой "npm run build" в папку dist.
